## Що це

Більшість туторіалів пояснюють права Linux через стіни тексту і таблиці, які треба зазубрювати. Ця сторінка працює інакше — ти натискаєш на символ і одразу бачиш що він означає.

Зроблено спеціально для україномовної аудиторії, яка тільки знайомиться з Linux або хоче нарешті розібратись з тими цифрами в `chmod`.

## Що всередині

- **Розбір рядка** — натисни на будь-який символ `-rw-r--r--` щоб побачити пояснення
- **Конструктор** — вибирай галочками які права хочеш, отримуй готову команду `chmod`
- **Octal-конвертер** — вводиш цифри `644`, бачиш що вони означають у символах
- **Таблиця прикладів** — найпоширеніші комбінації прав з поясненнями та security-тегами
- **Cheatsheet** — швидка шпаргалка по найважливіших `chmod` командах
